# Inventory Max

The server-side companion to **[Inventory Plus](https://modrinth.com/project/inventory-plus)**. It adds the inventory features that need a server's cooperation — so they work in multiplayer and stick around — and nothing else. Purely additive: it never changes how Inventory Plus already behaves, it only adds on top.

## Features

- **Container Locks** — lock a slot inside a placed chest, shulker, or barrel so sorting, automation, and stray clicks skip it. The lock is shared, so every player on the server sees it, and on a shulker it travels with the box when you break and replace it.
- **Equipment Slots** — dedicated slots for an elytra and a totem, so you can keep both ready without giving up your chestplate or a hand.
- **Pockets** — server-persistent pocket slots attached to your hotbar that you cycle in and out on demand. Real extra space, not a reshuffle of what you already have.

## Requirements

- **Inventory Plus** — Inventory Max is its companion and does nothing on its own.
- **MenuKit: Containers** (which brings in MenuKit).
- Fabric API, Minecraft 1.21.x. Universal — install on both client and server.
